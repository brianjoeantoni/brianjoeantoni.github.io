Pet Hotel: WOOF! Application Prototype

Pet Hotel: WOOF! is a prototype application that creates an opportunity for pet owners to board their pet(s) when they want to go out of
town or just simply does not have the time to care or groom their pet. This prototype application is made and designed by using figma.
I played the role as the head of UI/UX manager of the team in developing this prototype






BersihLah Android Application

BersihLah is an Android mobile application that allows the user to order cleaning services that are fully customisable to meet their specific
needs within the available categories. This application is made using Android Studio with Java, XML and firebase. I played the role of
designing the UI and code some part of the app using java.






A Review of Optical Text Recognition from Distorted Scene Image

A Review of Optical Text Recognition from Distorted Scene Image is a Research project that I did during my studies in Research Methododology
in Computer Science. This research paper explores and explains clearly about Text recognition as well as the algorithms that are used in recognizing
the text on different environments.






HOOPS Universe

HOOPS Universe is a web-based online store prototype that sells shoes and sport accessories. This Web-based online store is made by using
HTML, CSS and JavaScript. I played the role of designing the UI and integrating the UI with JS.
